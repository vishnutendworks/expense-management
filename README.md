# Expenses Management System

This is my Expense Management project developed using React, TypeScript, Vite, and Tailwind CSS. The main purpose of this project is to help users manage their daily or company expenses in an easy and organized way.

This project includes features like adding expenses, viewing expense details, category-wise tracking, dashboard analytics, and responsive UI design.

---

# Project Overview

The Expenses Management System is a frontend application created to simplify expense tracking and management.

In this project, users can:
- Add new expenses
- View expense details
- Track expenses based on categories
- Monitor spending through dashboard charts
- Manage claims and approvals UI

I mainly focused on creating a clean UI, responsive design, and better user experience.

# Installation & Setup

## Clone the Repository

```bash
git clone https://github.com/yourusername/expenses-management.git
```

## Move to Project Folder

```bash
cd expenses-management
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:5173
```

# State Management

For state management, I used React Context API to share data between components.

It helps to:
- Manage global expense data
- Avoid prop drilling
- Update UI dynamically

---

# Responsive Design

The application is fully responsive and works on:
- Mobile Devices
- Tablets
- Laptops
- Desktop Screens

Tailwind CSS responsive classes are used for building adaptive layouts.
